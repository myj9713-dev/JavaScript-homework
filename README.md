# 제출 안내

- 항목 별, 과제를 수행하고 과제 내용을 요약해 커밋(commit)합니다. (과제 갯수 = 커밋 갯수)
- 과제 수행 내용 및 회고를 README 파일에 작성하고 커밋합니다.
    - 과제 수행 과정 기록
    - 테스트 코드 작성
    - 과제 코드 리뷰
    - 회고 (깨달은 점, 아쉬운 점 등)
- 완료된 과제 결과를 자신의 과제 수행용 GitHub 저장소에 푸시(push)합니다.
- GitHub 저장소 URL을 과제 [스프레드시트]에 추가해 제출합니다.

### 제출 마감

2026년 1월 11일 오후 10시까지 [스프레드시트] 자신의 이름에 URL을 남깁니다.

### **평가 기준**

- 변수/상수 선언의 적절성 (const / let 구분)
- 조건문 활용의 적절성
- 함수 구현의 정확성
- 코드 가독성 (변수명, 들여쓰기)
- 주석 작성 (필요한 부분에 설명 추가)
