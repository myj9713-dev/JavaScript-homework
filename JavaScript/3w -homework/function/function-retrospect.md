## 테스트 코드 작성
각 로직이 의도대로 동작하는지 확인하기 위해 모든 변수를 `console.log()`로 직접 출력하여 검증.
1. **타입 검증**: `typeof`를 통해 모든 변수가 요구사항(문자열, 숫자, 불리언)에 맞게 선언됨을 확인.
2. **변경 및 보호**: `let` 변수는 재할당이 가능함을 확인하였고, `const` 상수는 재할당 시 에러가 발생하는 것을 확인하여 데이터 안정성을 검증함.
3. **가독성 확인**: 큰 숫자에 `_`를 사용하여 가독성을 높인 부분이 실제 출력 시 정상적인 숫자로 인식됨을 확인.
