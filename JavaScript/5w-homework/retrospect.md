# 과제 수행 내용 및 회고
## 과제 수행 과정 기록
1. **데이터 성격 분류**
- dashboard, tabButtons, tabPanels 등 DOM에서 한 번 찾아온 요소들은 프로그램 실행 중 참조 대상이 바뀌지 않으므로 const를 사용하여 의도치 않은 재할당을 방지함.

- targetId와 같이 클릭 시점마다 새로 계산되는 값 역시 해당 블록 내에서는 변하지 않으므로 const를 적용함.

2. **명명 규칙 수립**
- 변수명은 역할을 직관적으로 알 수 있도록 복수형(tabButtons)과 단수형(button)을 구분하여 명명함.

3. **가독성 최적화**
- 중첩된 for 문 대신 forEach 메서드를 사용하여 배열 요소에 직접 접근함으로써 코드의 길이를 줄이고 논리 흐름을 개선함.

- 아직 배우지 않은 getAttribute, setAttribute를 사용해 data-*속성을 가져와 사용.
